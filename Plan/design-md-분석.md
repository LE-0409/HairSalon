## 후보별 평가

### 1. **Airbnb** ⭐ 추천
- 흰 캔버스 + Airbnb Rausch(`#ff385c`) 단일 액센트 — 예약 CTA에 그대로 사용 가능
- **사진 중심**: 카드가 사진의 액자 역할만 함 (가장 큰 강점)
- **부드러운 곡선**: 카드 14px radius, 검색바·버튼 full pill
- "Airbnb Cereal" 친근한 모던 산세리프 — 살롱 톤과 잘 어울림
- 예약 마켓플레이스의 표준 UX 패턴(검색 → 카드 → 상세 → 예약)을 그대로 차용 가능
- 평가: **A ●○○** (캔버스가 흰색이라 따뜻함은 약함) / B ●●● / C ●●● / D ●● / E ●●●
- **종합: 8.0 / 10 — 가장 안전하고 검증된 선택**

### 2. **Mastercard** ⭐ 대안
- **퍼티-크림(`#F3F0EE`) 캔버스 = 따뜻함의 정수**
- 거대 라운드(40pt hero / 카드 full pill / 인물 원형 크롭)
- 인물 사진을 원형으로 자르는 패턴은 **디자이너 프로필**에 직접 활용 가능
- 시그널 오렌지를 살롱 액센트(예: 따뜻한 테라코타)로 치환하면 그대로 살롱 무드
- 에디토리얼 매거진 톤 — 고급 살롱에 매우 잘 맞음
- 평가: A ●●● / B ●● / C ●●● / D ●●● / E ●●
- **종합: 8.5 / 10 — 따뜻함·에디토리얼 톤이 최고**

### 3. **Starbucks**
- 크림 캔버스(`#f2f0eb`)와 따뜻한 톤 — 매우 잘 맞음
- 단점: 시그니처 그린이 워낙 강해 살롱 무드로 치환 시 정체성이 흔들림
- 카드 12px, 50px pill 버튼 — 곡선 좋음
- 평가: A ●●● / B ●● / C ●●● / D ●● / E ●●
- **종합: 7.0 / 10 — 색만 바꾸면 좋지만 식별성이 약해짐**

### 4. **Clay**
- 다채로운 단색 카드 + 클레이메이션 일러스트 — 살롱과는 톤이 다름
- "재미있는 GTM 제품" 무드라 신뢰감 있는 살롱 이미지와 충돌
- 평가: A ● / B ● / C ●● / D ● / E ●

### 6. **Apple**

- **종합: 6.0 / 10**
